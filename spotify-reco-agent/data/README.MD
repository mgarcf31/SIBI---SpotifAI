# Dataset

Por motivos de tamaño y licencia, el dataset no se incluye en el repositorio.

## Estructura esperada
Coloca aquí los archivos de datos antes de ejecutar los scripts:

- tracks.csv
- artists.csv
- genres.csv (si aplica)

Estos ficheros se utilizan únicamente para:
- Construir el grafo en Neo4j
- Generar los embeddings de las canciones

Una vez creada la base de datos, los datos no son necesarios para ejecutar la aplicación.
